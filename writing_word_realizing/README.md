###人工神经网络完成字体的分类鉴定

1、类比于MNIST手写数字集的识别，完成汉字的不同字体的分类识别

2、汉字的字体识别从字体库中获取

3、每个字体可以获取114个字，尽量包含汉字中的笔画的所有变化

4、使用卷积神经网络，对汉字图片进行基本的特征提取

5、标签类似于MNIST手写数字集的处理，为不同的字体的名称（宋体，黑体，等。。。）
 
6、完成训练分类识别
